# Recruit-List
산업기능요원 채용 공고 정리 사이트

## 시스템

1. 병무청에서 병역 지정업체 검색

2. 업체정보&채용정보를 채용관련 사이트에서 크롤링

### 스케줄링 주기

  매주 일요일 자정

## 기술스택

  Server
    
    * AWS EC2

    * MongoDB 4.0.16

    * MariaDB 10.4

  BackEnd

    * Java 1.8

    * Spring 5.2.3

    * Junit 5

    * JPA, Hibernate

  FrontEnd

    React
